# retrowave-frontend

## Project setup
```
yarn install
```

### Env variables
Create `.env` file. Required variables can be find in `.env-example` file


### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
